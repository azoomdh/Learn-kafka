tải về oracle DB

```bash
docker pull container-registry.oracle.com/database/free:latest
```

<br/>

chạy oracle DB

```bash
docker run -d \
  --name azoo \ # tên DB là azoo
  -p 1521:1521 \
  -p 5500:5500 \
  -e ORACLE_PWD=a0000 \ # mật khẩu là a0000
  container-registry.oracle.com/database/free:latest
```
