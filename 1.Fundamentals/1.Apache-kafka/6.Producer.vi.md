> https://developer.confluent.io/courses/apache-kafka/producers/

<br/>

producer = client gửi message vào các topic kafka

broker = quản lý lưu trữ các phân vùng, quản lý nhân bản

<br/>

sử dụng kafka producer api

<br/>

file `kafka.properties`

```properties
bootstrap.servers = broker1, broker2, broker3
acks = all
```

file ví dụ

```java
Properties config = loadProperties("kafka.properties")

Producer<Long, String> producer = new KafkaProducer<>(config);

TemperatureData_Entity temperatureData = new TemperatureData_Entity("phong bep", 32, Temperature.celcius(22) );
//                                                                     ^ các trường của hàng (bao gồm id=32)
String temperatureData_StringJson = objectMapper.writeValueAsString(temperatureData);

ProducerRecord temperatureData_ProducerRecord = new ProducerRecord<>("TemperatureData_Topic, 32, temperatureData_StringJson");
//                                                                     ^ tên topic            ^id hàng        ^ dữ liệu của hàng

producer.send(temperatureData_ProducerRecord);
```

<br/>

`temperatureData_StringJson` chỉ là mảng byte với kafka

phải dùng thư viện để giải tuần tự hóa
